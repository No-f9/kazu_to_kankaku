# かずパッと 👀🔴🟡

かぞえないで、パッと見て「かず」を実感する力を遊びで育てる、子ども向け（2〜8歳むけ）のウェブアプリです。
**5までは一目で分かる／6以上は「5といくつ」でとらえる**という考え方をもとに、数の理解・多少・たし算（合成）・ひき算（分解）を、玉（たま）と数式で行き来しながら身につけます。

ファイルを開くだけで動く **単一HTML**（インストール・ネット接続不要）。スマホ・タブレット・パソコンのブラウザで遊べます。

## 公開ページ

GitHub Pages を有効にすると、次のURLで公開されます（`ユーザー名`と`リポジトリ名`はご自身のものに置き換わります）：

```
https://ユーザー名.github.io/リポジトリ名/
```

## あそびかた

- **いくつ かな？** … トレイの玉が何個かを、テンキーで答える（瞬時に数を実感）
- **どっちが おおい？** … 多い方のトレイをタップ（多少の感覚）
- **あわせて いくつ？** … たし算（合成）。答え合わせで `7 + 1 = 8` を全部の数の玉で表示
- **のこりは いくつ？** … ひき算（分解）。答え合わせで `7 − 3 = 4` を全部の数の玉で表示

設定：**かずのはんい**（〜5／〜10／〜20）、**レベル**（🔴たま＝玉を見せる／🔢すうしき＝数式だけ→正解で玉を表示）。
回答はテンキー（電卓方式）。答え合わせは効果音のみ（🔊でオン・オフ）。玉は5のまとまりが分かるよう赤・黄で色分け、トレイは範囲の上限まで空マスで表示します。

## がんばりカレンダー（ハビットトラッカー）

毎日の取り組みを記録できます。`📅` ボタンでカレンダーを開けます。

- **1日の目標問題数**を設定（カレンダー下の `−` / `＋`、1〜30問）
- 目標数だけ正解すると、その日に **💮 はんこ** が押されます
- 目標達成の瞬間に「**たっせい おめでとう！**」のお祝いが出ます
- 月を `◀` `▶` でめくると、**過去の達成履歴**（はんこ）や、今月の達成日数・連続日数も見られます

※記録はお使いの端末・ブラウザに保存されます（同じ端末・同じブラウザで開けば続きから見られます）。

## 公開のしかた（ブラウザだけ・コマンド不要）

1. GitHub で新しいリポジトリを **Public** で作成
2. 「Add file → Upload files」で `index.html` をアップロードして Commit
3. リポジトリの **Settings → Pages** を開く
4. **Source** を「Deploy from a branch」、**Branch** を `main` / `/(root)` にして **Save**
5. 1分ほど待つと、ページ上部に公開URLが表示されます

## クレジット

数の感覚を育てる一般的な算数教育の考え方（5進法による数のまとまり、瞬時把握、合成・分解など）をもとに、デザイン・実装はすべてオリジナルで制作しています。特定の市販教具・商標とは無関係です。
